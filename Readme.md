# Data Cleaning & Analysis Project

##  Overview
This project focuses on cleaning and analyzing customer shopping behavior data using Python and SQL.

##  Tools Used
- Python (Pandas)
- SQL (SQLite)
- Power BI (for visualization)

##  Steps Performed
- Data cleaning using Pandas
- Handling missing values
- Feature transformation
- SQL queries for analysis

##  Output
Clean dataset ready for Power BI dashboard

##  How to Run
1. Install dependencies
2. Run cleaning.py
